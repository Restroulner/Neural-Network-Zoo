# Neural Network Zoo

This repository contains implementations of various neural network architectures from scratch, primarily using Python and popular deep learning frameworks like TensorFlow and PyTorch.

## Architectures Included:

- **Perceptron**: A basic building block of neural networks.
- **Multi-Layer Perceptron (MLP)**: Fully connected neural networks for classification and regression.
- **Convolutional Neural Network (CNN)**: Architectures for image processing tasks.
- **Recurrent Neural Network (RNN)**: Basic RNN, LSTM, and GRU for sequential data.

## Getting Started

To explore the implementations, clone the repository:

```bash
git clone https://github.com/Restroulner/Neural-Network-Zoo.git
cd Neural-Network-Zoo
```

Each architecture will have its own directory with a dedicated `README.md` and Python implementation.
